GitHub Student

O Github oferece o Github Pro gratuito pela duração do curso, assim como acesso a versão de desenvolvedor do Copilot, acesso a cursos, certificações e ofertas de projetos open-source para formação de portfolio. Além de ofertas especiais de mais 40 parceiros, incluindo plataformas de ensino, dominio gratis e serviços de infraestrutura.

A Cisco possui a sua propria plataforma de ensino, enquanto alguns cursos são pagos, existem varias opções gratuitas com certificação em diversas areas. Dio

A Dio é uma plataforma de vagas voltada para T.I, ela possui um plano pago com acesso a centenas de cursos, mas também oferece cursos gratuitos com certificação, além de oferecer Bootcamps em parcerias com diversas empresas, esses bootcamps são muitas vezes gratuitos e usados pelas empresas parceiras para busca de talentos.

FreeCodeCamp

O FreeCodeCamp é uma organização sem fins lucrativos internacional, que busca formar desenvolvedores Full-Stack, ela possui um programa robusto de ensino, com cursos de front, back, analise de dados entre outras funções, fornece certificado e mais de 100.000 graduados conseguiram empregos na industria.
